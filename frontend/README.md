# 🚀 PromptCatalyst
PromptCatalyst is an AI-powered Chrome extension that transforms simple prompts into well-structured, optimized prompts for various Large Language Models such as ChatGPT, Gemini, Claude, DeepSeek, and more.

It leverages Google's Gemini API to intelligently rewrite prompts, helping users receive higher-quality AI responses with minimal effort.
## Features

- Optimize prompts using Google Gemini AI
- Support for multiple target AI models
- Clean and responsive React UI
- Express.js backend
- REST API architecture
- Copy optimized prompts instantly
- Fast and lightweight
