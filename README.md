# PromptCatalyst

PromptCatalyst is a Chrome extension project that helps users turn rough prompts into polished, optimized AI inputs for tools like ChatGPT, Gemini, Claude, and other LLMs.

## Project Structure

- `frontend/` – React + Vite UI for the extension
- `backend/` – Express.js API that handles prompt optimization requests
- `postman/` – API collection and mock/test assets

## Tech Stack

- React
- Vite
- Express.js
- Node.js
- Google Gemini API

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   cd frontend && npm install
   cd ../backend && npm install
   ```
2. Start the frontend:
   ```bash
   cd frontend && npm run dev
   ```
3. Start the backend:
   ```bash
   cd backend && npm run dev
   ```
